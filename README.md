## Advent Of Code 2025
Uma tentativa de resolver os dois primeiros dias do Advent Of Code 2025 sem ainda ter estudado DSA (Estrutura de Dados e Algoritmos). 

No momento que criei esse repositório, estou focado em aprender Java através do [Java Programming MOOC](https://java-programming.mooc.fi/) da Universidade de Helsinki, e apesar do curso abordar um [básico de algoritmos na parte 7](https://java-programming.mooc.fi/part-7), atualmente estou na parte 5 e *ainda* não peguei para estudar o assunto separadamente.

O objetivo com isso é ver como me saio tentando resolver os dois primeiros dias sem ainda ter esse conhecimento em estrutura de dados e algoritmos. E da forma que eu resolver (se resolver) irá ficar do jeito que está. Ou seja, mesmo que futuramente eu já tenha estudado o assunto, não voltarei aqui para melhorar a solução, dessa forma poderei ver minha evolução nos anos subsequentes.
